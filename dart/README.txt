**** FLIGHT FLIPPER ****
************************

***** USER STORY *****
* The user will see a clear logo upon entering site
* The user will be given clear instructions on what to do next
* The user will see a form with simple input fields
* The user will see a clear 'submit' button
* Upon clicking the submit button, the user will see a new page materialize, making it clear that the button click worked
* If the submit button fails, there will be a message saying why
* Once the user successfully enters information, they will see a screen with their specific results
* The user will see results that are ordered from price lowest to highest
* The results will be image heavy, and simple have the name of the destination, the price, and a button to purchase the flight.
